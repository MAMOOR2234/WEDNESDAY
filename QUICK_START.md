## OPTIONAL - Switch to OpenAI (5 Minutes)

### Step 1: Get OpenAI API Key (2 min)
```
1. Visit: https://platform.openai.com/api-keys
2. Click "Create new secret key"
3. Copy the key (looks like: sk-proj-xxx...)
4. Keep it safe!
```

### Step 2: Update .env (1 min)
Open `.env` file in WEDNESDAY folder and change:

```env
# Change this:
BRAIN_TYPE=gemini

# To this:
BRAIN_TYPE=openai
OPENAI_API_KEY=sk-proj-your-key-here
OPENAI_MODEL=gpt-4o-mini
```

### Step 3: Install OpenAI (1 min)
```bash
pip install openai
```

### Step 4: Run Wednesday with OpenAI (1 min)
```bash
python main.py
```

**Done!** Now Wednesday uses ChatGPT. 🤖
